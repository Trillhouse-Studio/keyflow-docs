---
title: Credentials
description: Learn about the integration and third party credentials in Keyflow.
---

Credentials page contains all the third party integrations provided by Keyflow.

Click on the `Add API Key` to connect the integration with keyflow using your API Key. Click on `Connect` to login and connect into any service of your choice.

After you are done using the integration/service, you can `Revoke` the connection later on.

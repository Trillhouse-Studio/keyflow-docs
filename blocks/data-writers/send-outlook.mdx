## Overview

Send emails from your Microsoft Outlook account, enabling efficient management of email-based workflows and ensuring important communications are not missed. This block also supports savings emails as draft in your Outlook account.

<Note>
  To run this block, you will first need to authenticate your Microsoft Outlook
  account with Keyflow. Navigate to your
  [credentials](https://keyflow.space/settings/credentials) page to connect.
</Note>

## Block Inputs

| Name              | Type    | Description                                                |
| ----------------- | ------- | ---------------------------------------------------------- |
| `recipient_email` | string  | The email address of the recipient.                        |
| `email_body`      | string  | The content of the email message.                          |
| `email_subject`   | string  | The subject of the email.                                  |
| `message_id`      | string  | Message ID of an email to reply to.                        |
| `send_as_draft`   | boolean | When enabled, the email is saved as "Draft" in your gmail. |

## Block Outputs

| Name           | Type   | Description                                                       |
| -------------- | ------ | ----------------------------------------------------------------- |
| `email_status` | string | Reports on the success or failure of the email sending operation. |

## Use Cases

1. **Regular Updates:** Automate the process of sending regular updates or reports to stakeholders.
2. **Trigger-based Workflows:** Create a workflow to respond to certain triggers by sending out emails.
3. **Help Desk Systems:** Implement a help desk system that requires sending acknowledgments, replies, or follow-up messages to users.
4. **Customer Service and Marketing Automation:** Automate routine email correspondence to enhance user experience and engagement.

The OutlookSender node is particularly useful for automating routine email correspondence, enabling you to save time and ensure consistent communication patterns.

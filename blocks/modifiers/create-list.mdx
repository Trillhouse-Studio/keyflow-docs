## Overview

Create a list from multiple seperate inputs and get a unified structure.

## Block Inputs

| Name      | Type | Description                |
| --------- | ---- | -------------------------- |
| `input_1` | any  | First element of the list  |
| `input_2` | any  | Second element of the list |

<Note>
  To build a list of more than 2 inputs, You can clik the "+" or "-" icon below
  the input name to add or remove new input handles that you can connect other
  blocks too.
</Note>

## Block Outputs

| Name   | Type   | Description                                          |
| ------ | ------ | ---------------------------------------------------- |
| `list` | any[ ] | A list of all the seperate inputs combined into one. |

## Use Cases

1. **Batch Processing:** Particularly useful when needing to create a single list to process in batch mode.
2. **Writing data:** Blocks like Write to Google Sheets etc required a list of data to be provided in order to process data.

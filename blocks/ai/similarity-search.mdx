title: Similarity Search
---

## Overview

Search through large bodies of text by breaking it down and finding the most relevant parts based on a search query.

## Block Inputs

| Name          | Type   | Description                                                  |
| ------------- | ------ | ------------------------------------------------------------ |
| `query`       | string | A search term or phrase to search in the text.               |
| `num_results` | number | Number of relevant text sections to be retrieved.            |
| `chunk_size`  | number | Size of each text section in tokens. Default is 1000 tokens. |
| `text`        | string | The large body of text you want to search within.            |

## Block Outputs

| Name              | Type      | Description                                      |
| ----------------- | --------- | ------------------------------------------------ |
| `relevant_chunks` | string[ ] | The most relevant sections to your search query. |

## Use Cases

1. **Researching:** Quickly finding specific information within substantial amounts of text.
2. **Filtering Reports:** Searching for specific details within extensive reports.
3. **Extracting Key Sections:** Rapidly extracting essential sections from long articles.
